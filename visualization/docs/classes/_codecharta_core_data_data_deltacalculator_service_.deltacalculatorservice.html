fillMapsWithNonExistingNodesFromOtherMap

  • fillMapsWithNonExistingNodesFromOtherMap(leftMap: CodeMap, rightMap: CodeMap): object
    • Defined in codeCharta/core/data/data.deltaCalculator.service.ts:60

    Unifies both maps and inserts unique nodes into the other map. The inserted nodes have all metrics set to zero. This will calculate deltas correctly between versions and add "empty" buildings to ensure all revisions have an equal building pool. These empty buildings should be viewed with the unary metric as area
